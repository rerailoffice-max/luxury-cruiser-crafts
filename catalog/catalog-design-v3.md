# KOGEI CODE カタログ設計書 v4.0

> **v4.0 設計方針:**
> - コンセプト: **オールジャパン — 船体から工芸まで、すべて日本製**
> - Phase 1 メイン船: **Yamaha EXULT 43**（43ft / 13.98m）— 日本最高峰フラッグシップ
> - フラッグシップ: **Lexus LY680**（68ft / 20.66m）— 日本が世界に誇るラグジュアリー
> - PART 1 に NOT A HOTEL 風の「所有モデル＋利益スキーム」セクションを新設
> - 体験価値をメインに訴求（寿司職人・和牛BBQ・伝統工芸空間）
> - Azimut Grande 27M は参考ラインナップとして1枚のみ掲載
> - 総スライド数: **30枚**
> - 3/6会議決定を全面反映

## デザインシステム

### カラーパレット
| 色 | HEX | 用途 |
|----|-----|------|
| メインブラック | `#1A1A1A` | 背景・テキスト |
| ゴールド | `#C9A962` | アクセント・見出し・引き出し線 |
| ゴールド（淡） | `#E8D5A3` | サブアクセント |
| ホワイト | `#FFFFFF` | テキスト・背景（交互） |
| ディープレッド | `#8B0000` | ワンポイント（漆イメージ） |
| ネイビー | `#0A1628` | MARINA MAP 背景 |

### タイポグラフィ
| フォント | 用途 |
|----------|------|
| **Playfair Display** | 英文見出し |
| **Noto Sans JP** Light/Regular | 日本語本文・キャプション |
| **Noto Serif JP** | 引用・コンセプトテキスト |

### 内装スプレッドの統一レイアウト
- ページの **60-70%**: 大きなAI生成内装イメージ
- イメージ上に **Gold引き出し線** で各部位をポイント
- 各ポイントに **工芸名 / 産地 / 特徴1行** のキャプション
- ページ下部 or 右端: 工芸素材の **ミニクローズアップ写真**（2-3枚）
