# 富裕層向けクルーザー × 日本伝統工芸 カタログ設計書 v3

> **v3 設計方針:**
> - CRAFTS / CONCEPTS セクションを廃止 → 工芸は船の内装ページに統合
> - SERVICE を序盤に配置（投資家＋オーナー両面のメリット訴求）
> - MARINA MAP → 各マリーナ × 船の内装詳細、という体験型フロー
> - 2マリーナ × 2メイン船 + その他4モデル
> - 総スライド数: **33枚**

---

| 項目 | 内容 |

## デザインシステム

### カラーパレット
| 色 | HEX | 用途 |
|----|-----|------|
| メインブラック | `#1A1A1A` | 背景・テキスト |
| ゴールド | `#C9A962` | アクセント・見出し・引き出し線 |

| ディープレッド | `#8B0000` | ワンポイント（漆イメージ） |
| ネイビー | `#0A1628` | MARINA MAP 背景 |

### タイポグラフィ
| フォント | 用途 |
|----------|------|
| **Playfair Display** | 英文見出し |
| **Noto Sans JP** Light/Regular | 日本語本文・キャプション |
| **Noto Serif JP** | 引用・コンセプトテキスト |

### 内装スプレッドの統一レイアウト
- ページの **60-70%**: 大きなAI生成内装イメージ
- イメージ上に **Gold引き出し線** で各部位をポイント
- 各ポイントに **工芸名 / 産地 / 特徴1行** のキャプション
- ページ下部 or 右端: 工芸素材の **ミニクローズアップ写真**（2-3枚）
