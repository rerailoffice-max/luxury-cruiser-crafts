# クルーザー投資事業 スキーム調査報告書

> **調査目的:** NOT A HOTELのビジネスモデルを参考に、クルーザーを投資家に貸し出す事業の法的要件・許認可・座組を調査する。
> **調査日:** 2026年3月6日（3月6日会議決定を反映し更新）
> **基本方針:** オールジャパン（日本製船舶 × 日本の伝統工芸）。Phase 1はクローズド募集（知り合い経由）。
> **関連資料:** [クルーザー購入_リフィット調査報告書.md](クルーザー購入_リフィット調査報告書.md) / [クルーザー投資_ターゲット市場調査.md](クルーザー投資_ターゲット市場調査.md)

## 1. エグゼクティブサマリー

### 結論

クルーザーを投資家に貸し出す事業は、スキームの選択により必要な許認可が大きく異なる。

| スキーム | 第二種金融商品取引業 | 海上運送関連 | 初期コスト | 推薦度 |
|---------|-------------------|------------|----------|--------|
| **A. 共同所有型** | 不要（条件付き） | 船舶貸渡業届出 | 低 | ★★★★★ |
| **B. ファンド型** | **必要** | 旅客事業許可 | 高 | ★★★★ |
| **C. サブスク会員型** | 不要 | 船舶貸渡業届出 | 中 | ★★★ |

### 推薦

**Phase 1 は A案（共同所有型）でスタートし、実績を積んだ後に B案（ファンド型）へ移行する。**

理由:
1. A案は第二種金融商品取引業の登録が不要で、早期にビジネスを開始できる
2. 船舶の共同持分を直接登記する方式のため、法的リスクが低い
3. 既にYacht Life Designが同様のモデルで事業展開中、先行事例がある
4. Phase 2 で第二種を取得し、ファンド型で投資家を広く募集する展開が可能

## 2. NOT A HOTELのビジネスモデル分析

### 基本構造

NOT A HOTELは「住宅 × ホテル × 投資」を融合させたハイブリッド型プラットフォーム。

```
┌─────────────────────────────────────────────┐
│              NOT A HOTEL グループ              │
│                                              │
│  NOT A HOTEL 本体        NOT A HOTEL 2ND     │
│  ・物件の企画/販売        ・第二種金融商品取引業  │
│  ・ホテル運営/管理        ・セカンダリーマーケット │
│  ・サブリース運用         ・買取/再販仲介         │
│                                              │
│  NOT A HOTEL DAO                             │
│  ・暗号資産 NAC 発行                           │
│  ・RWA（実物資産トークン化）                     │
└─────────────────────────────────────────────┘
```

### シェア購入の仕組み

| 項目 | 内容 |
|------|------|
| **最小単位** | 1/36の所有権（年間10泊分） |
| **標準単位** | 1/12の所有権（年間30泊分） |
| **権利形態** | 実際の「所有権」（利用権や会員権ではない） |
| **資産性** | 減価償却・譲渡・相続が可能 |
| **売却** | 購入4年目以降、セカンダリーマーケットで売却可能 |

### 収益構造（NOT A HOTEL側）

| 収益源 | 内容 | 割合/金額 |
|--------|------|----------|
| **販売利益** | 完成前にオンラインで販売、利益確定後に建設 | 売上のメイン |
| **管理費** | 販売金額の3%を月額で継続徴収 | 例: 3,000万円購入 → 年90万円 |
| **サブリース利益** | オーナー未使用日を買い取り、NFT宿泊チケットとして一般ゲストに販売 | 運用益のメイン |
| **NACコイン** | 暗号資産 NOT A HOTEL COIN の販売・流通 | IEOで約20.7億円調達 |

### オーナーのメリット

1. **未使用泊の買取**: 使わない泊数をNOT A HOTELが固定収益で買い取り、管理費と相殺可能
2. **全国相互利用**: 全国のNOT A HOTEL施設を相互利用（同ランク以下は追加費用なし）
3. **手間ゼロ**: 清掃・維持管理は全てNOT A HOTELが負担
4. **資産価値**: 所有権のため減価償却・売却・相続が可能

### 法的ポイント

- NOT A HOTEL 2ND Inc. が**第二種金融商品取引業**（関東財務局長（金商）第3269号）を取得
- 資本金 1億2,850万円
- セカンダリーマーケット（二次取引プラットフォーム）の運営にこの登録が必要
- 一部物件では不動産を**信託受益権化**して販売

### 資金調達実績

| ラウンド | 金額 | 時期 |
|---------|------|------|
| シード | 約9.3億円 | 初期 |
| シリーズA | 約28.5億円 | 2023年2月 |
| シリーズB | 約44億円 | 2024年7月 |
| IEO（NACコイン） | 約20.7億円 | 2024年12月 |
| デット（銀行借入） | 105億円 | 2024年12月 |
| **累計** | **約223億円** | |

## 3. 第二種金融商品取引業の登録要件

### いつ必要か？

| ケース | 第二種の要否 |
|--------|------------|
| 匿名組合契約で投資家から出資を募る | **必要** |
| ファンド型で小口化して投資商品として販売 | **必要** |
| セカンダリーマーケット（二次売買仲介）を運営 | **必要** |
| 信託受益権を販売 | **必要** |
| 船舶の共同持分を直接売買（所有権の売買） | 不要の可能性が高い |
| 会員制レンタル（利用権の販売） | 不要 |

### 登録要件

| 項目 | 要件 |
|------|------|
| **法人の場合** | 最低資本金 **1,000万円** |
| **個人の場合** | 営業保証金 **1,000万円**（供託所に供託） |
| **登録免許税** | **15万円** |
| **標準処理期間** | 約2ヶ月（事前相談・補正期間を除く） |
| **実際の所要期間** | **6ヶ月〜1年**（事前相談から業務開始まで） |

### 人的要件

| 役職 | 要件 |
|------|------|
| **経営者（代表取締役）** | 金融商品取引業・リスク管理の知識と経験 |
| **コンプライアンス責任者** | 常勤・兼務不可、金融機関での実務経験が望ましい |
| **内部管理責任者** | 営業部門から独立した管理部門に配置 |
| **最低人数** | 3名以上が目安 |

### 組織体制の要件

- 営業部門から独立した**コンプライアンス部門**の設置が必須
- 帳簿書類管理、リスク管理、顧客管理、広告審査、苦情処理、内部監査の体制
- ADR（裁判外紛争解決）措置の整備
- 第二種金融商品取引業協会への加入（加入しない場合は同等の社内規則整備が必要）

### 登録手順

```
① 事前相談（関東財務局 048-600-1293）
   └─ 業務スキーム・営業方法・組織体制を説明
     ↓
② 申請書提出（金融庁電子申請・届出システム）
   └─ 登録申請書、定款、組織図、役員履歴書、
      事業計画書、収支見込み（3年分）等
     ↓
③ 審査（財務事務所 → 本局）
   └─ 標準2ヶ月（補正期間含まず）
     ↓
④ 登録・登録済通知書送付
     ↓
⑤ ADR措置・協会加入
     ↓
⑥ 業務開始
```

### 登録にかかる費用の目安

| 項目 | 金額 |
|------|------|
| 登録免許税 | 15万円 |
| 行政書士・弁護士報酬 | 200万〜500万円（スキームの複雑さによる） |
| 資本金 | 1,000万円以上 |
| 人件費（コンプライアンス担当等） | 年間600万〜1,000万円 |
| システム整備費 | 100万〜300万円 |
| **合計（初年度）** | **約2,000万〜3,000万円** |

## 4. クルーザー事業に必要なその他の許認可

### 許認可の全体マップ

| 事業内容 | 必要な許認可 | 管轄 | 難易度 |
|---------|------------|------|--------|
| 投資家から資金を集める（ファンド型） | 第二種金融商品取引業 | 金融庁/関東財務局 | ★★★★★ |
| 船を貸し出す（裸用船） | 船舶貸渡業 届出 | 国土交通省 | ★ |
| クルーで旅客を運ぶ（13名以上） | 旅客不定期航路事業 許可 | 地方運輸局 | ★★★★ |
| クルーで旅客を運ぶ（小型船） | 内航不定期航路事業 登録 | 地方運輸局 | ★★★ |
| 旅行パッケージとして販売 | 旅行業登録 | 観光庁 | ★★★ |
| オーナーが自分で操縦 | 小型船舶操縦免許（1級） | 国土交通省 | ★★ |

### 旅客不定期航路事業の詳細

投資家やゲストを乗せてクルージングする場合に必要となる。

| 項目 | 要件 |
|------|------|
| **適用条件** | 旅客定員13名以上の船舶で不定期に旅客運送 |
| **許可申請先** | 地方運輸局長 |
| **取得期間** | 6ヶ月〜1年 |
| **必要人員** | 安全統括管理者 + 運航管理者（船長経験3年以上） |
| **罰則** | 無許可営業は3年以下の懲役 or 300万円以下の罰金 |

**注意:** 年間3日以内の運航は届出で対応可能。

### 船舶貸渡業の詳細

船そのものをオーナーに貸し出す場合に必要。

| 項目 | 内容 |
|------|------|
| **手続き** | 届出制（許可不要） |
| **届出先** | 地方運輸局 |
| **難易度** | 低（書類提出のみ） |

### A案: 共同所有型（推薦 — Phase 1）

Yacht Life Designと同様の方式。**第二種金融商品取引業の登録が不要**で、最も早く事業を開始できる。

```
┌─────────────────────────────────────────────┐
│           A案: 共同所有型スキーム              │
│                                              │
│  投資家A ─┐                                   │
│  投資家B ─┤─→ 船舶の共同持分を直接購入          │
│  投資家C ─┤   （船舶登記に共同オーナーとして記載）│
│  投資家D ─┘                                   │
│           │                                   │
│           ↓                                   │
│     運営会社（御社）                            │
│     ・メンテナンス管理                          │
│     ・利用スケジュール調整                       │
│     ・クルー手配                               │
│     ・保険手続き                               │
│           │                                   │
│           ↓                                   │
│     管理費（年会費）で収益確保                    │
└─────────────────────────────────────────────┘
```

### B案: ファンド型（Phase 2）

NOT A HOTELと同様の方式。投資家から匿名組合契約で資金を集め、運営会社がクルーザーを購入・運用する。

```
┌─────────────────────────────────────────────┐
│           B案: ファンド型スキーム              │
│                                              │
│  投資家（多数）                                │
│     │ 匿名組合出資                            │
│     ↓                                        │
│  ファンド運営会社（御社 or 子会社）              │
│  【第二種金融商品取引業 登録必須】               │
│     │                                        │
│     ├─→ クルーザー購入・管理                    │
│     ├─→ 利用権の販売（投資家 + 一般会員）        │
│     ├─→ チャーター運用（サブリース）             │
│     └─→ 収益分配（投資家へ）                    │
│                                              │
│  収益源:                                      │
│  ① ファンド組成手数料                          │
│  ② 管理費（資産の3〜5%/年）                    │
│  ③ チャーター収入の運営取り分                    │
│  ④ セカンダリー売買手数料                       │
└─────────────────────────────────────────────┘
```

**NOT A HOTELとの対比:**

| NOT A HOTEL | クルーザー版 |
|-------------|-----------|
| 別荘の所有権をシェア販売 | クルーザーの利用権をファンドで販売 |
| 未使用日をNFTチケットで販売 | 未使用日をチャーター運用 |
| 管理費 販売額の3% | 管理費 資産価値の3〜5% |
| セカンダリーマーケットで売却可 | 持分のセカンダリー売却可 |
| NOT A HOTEL 2NDが第二種を保有 | 御社（or子会社）で第二種を取得 |

**必要な許認可:**
- **第二種金融商品取引業**（関東財務局）
- 船舶貸渡業の届出
- チャーター運用する場合は旅客事業許可

**メリット:**
- 小口化できるため投資家を広く集められる
- 運営会社に裁量権があり効率的に運用可能
- NOT A HOTEL型の高い収益性

**デメリット:**
- 第二種取得に6ヶ月〜1年、費用2,000万〜3,000万円
- コンプライアンス体制の維持コスト
- 金融庁の監督下に入る

### C案: サブスク会員型（ライト版）

ヤマハシースタイルに近い方式。自社所有のクルーザーを会員に貸し出す。

```
┌─────────────────────────────────────────────┐
│           C案: サブスク会員型                  │
│                                              │
│  御社（クルーザーオーナー）                      │
│     │                                        │
│     ├─→ 年額/月額会員制で利用権を販売            │
│     ├─→ 会員がスケジュール予約して利用            │
│     └─→ クルー付きで運航                       │
│                                              │
│  会員種別:                                    │
│  ・プレミアム会員（年額500万円、月4回利用）       │
│  ・スタンダード会員（年額200万円、月2回利用）     │
│  ・ゲスト利用（1回50万〜100万円）               │
└─────────────────────────────────────────────┘
```

**必要な許認可:**
- 船舶貸渡業の届出
- 旅客を運ぶ場合は旅客事業許可

**メリット:**
- 第二種金融商品取引業が不要
- シンプルな事業構造

**デメリット:**
- 自社で船舶購入費を全額負担する必要がある
- スケールしにくい

## 8. 重要な法的注意事項

### 投資助言業との関係

| 業態 | 登録の要否 | 説明 |
|------|----------|------|
| **投資助言・代理業** | 御社の事業では通常不要 | 有価証券の投資判断を助言する場合に必要。クルーザーのシェア販売自体は投資助言に該当しない |
| **投資運用業** | B案で自ら資産運用する場合は必要の可能性 | 投資家から集めた資金でクルーザーを購入し運用益を分配する場合 |
| **第二種金融商品取引業** | B案で必要 | 匿名組合出資の募集・取扱いに該当 |

### 税務上の留意点

- **共同所有の場合**: 各オーナーが持分に応じた減価償却が可能
- **ファンドの場合**: 匿名組合の利益は出資者に帰属（パススルー課税）
- **船舶の耐用年数**: モーターボート 4年（中古の場合は簡便法で2年も可能）
- **消費税**: 船舶の購入・チャーター料に10%課税
