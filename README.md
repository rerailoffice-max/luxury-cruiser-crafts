## 現在のステータス

**フェーズ**: 方針転換後の再設計中（2026-04-15更新）

### 事業構成（3本柱）

#### A. KOGEI CODE 鑑定事業（最優先・新規）
- PSA社（トレカ鑑定30年）をモデルにした工芸品グレーディングシステム
- 1-10点評価、経産省認定5項目を100項目に拡張
- 鑑定料ビジネス（初期 3,000円程度）
- eBay販売チャネルでの鑑定証明書付き販売
- 将来: 感性工学（慶應大学）の要素組み込み、スニーカーダンク型プラットフォーム

#### B. クルーザー内装受注事業（検討中）
- 自社所有ではなく、購入者への内装工事請負モデル
- 日本に船舶内装専門業者が少ない（競合優位性）
- 年間維持費1,000万円の問題を回避

#### C. クルーザーパンフレット（継続配布）
- NOT A HOTELモデル: 配布して共感者・投資家を集め、実行に移す
- 修正: Lexus写真への変更、マリーナ位置調整

### 方針転換の経緯（2026-04-15）

| 項目 | 変更前（3/29時点） | 変更後 |
|------|-------------------|--------|
| 船舶所有 | アジムット社製、共同所有で購入 | 一旦棚上げ → 内装受注ビジネスに |
| 事業の主軸 | クルーザー体験 | KOGEI CODE鑑定事業 |
| 収益モデル | 共同オーナー会費 | 鑑定料 + eBay販売 |
| ターゲット | 富裕層（分割所有） | 工芸品コレクター・職人 |
| マリーナ | 横浜ベイサイド確定 | 事業用途不可が判明、要再検討 |

**転換理由:**
- ベイサイドマリーナ見学で年間維持費1,000万円と事業用途制限が判明
- 船舶購入前にキャッシュフローを作る必要がある
- 鑑定事業は投資が少なく、収益化が早い
