## プロジェクト概要

| 項目 | 内容 |
|------|------|
| **概要** | 日本の伝統工芸×クルーザーによる富裕層向けブランド事業 |
| **コンセプト** | 「海の上の禅寺」— 禅の思想に基づく引き算の美学と五感体験 |
| **ターゲット** | 富裕層・インバウンド（海外富裕層） |

## 現在のステータス

**フェーズ**: 内装デザインコンセプト確定 → AK社打ち合わせ準備

### 確定事項（2026-03-29時点）

- **船舶**: アジムット社製クルーザー（24名収容、12〜15億円）
- **内装コンセプト**: 「海の上の禅寺」
  - 禅の思想に基づく引き算の美学
  - 風水の四大元素（火・水・土・風）に基づく部屋分け
  - ゲストの好みに応じた元素別宿泊体験の提供
- **拠点**: 横浜ベイサイドマリーナ（第一候補として確定）
  - 台風・地震リスクが低い、富裕層集積地、海上保安庁連携体制あり
  - 三重県案は一旦保留（立地・自然災害リスク）
- **活用予定の伝統工芸**:
  - 日本刀（製作2-3年、約250万円）
  - 盆栽（ミイラ化技術による永続保存）
  - 組子細工
  - 彫刻
- **資金調達**: 分割所有方式 + マザーファンド（オープンスカイ社長経由）
- **事業実績作り**: 船完成前に工芸品販売事業を展開、「工芸コード」ブランド確立
