# LP参考サイト調査レポート

> **調査日:** 2026年3月7日
> **目的:** KOGEI CODE 共同オーナー募集LP制作に向けた参考サイト調査
> **方針:** 高級感 × 承認欲求 × 体験価値を軸にしたLP設計の根拠資料

---

## 調査サマリー

9つの参考サイトを5カテゴリに分類。今回のサービス（EXULT 43 / LY680 共同所有、オールジャパン戦略、体験価値重視、法人経営者＋開業医ターゲット）に対し、各サイトのデザイン手法・訴求構造を分析した。

## A. 共同所有 × 高級感

### 1. NOT A HOTEL

| 項目 | 内容 |
|------|------|
| URL | https://notahotel.com / https://notahotel.com/en/lp/nah |
| 業態 | 高級別荘のシェア購入 |
| 類似度 | ★★★★★ |

**デザイン分析:**
- 黒背景 × 大判フルスクリーン写真のヒーロー
- 「4つの特徴」をカード型で並列配置（シェア購入 / 資産保有 / 相互利用 / 維持管理不要）
- オーナーの声で「会食10回分の価値」など体験を言語化
- CTAは「資料請求」のみ — クローズド感を演出
- 物件カードに「価格 / 広さ / 定員」を端的に表示

**取り入れるべき要素:** 4特徴カード構成、オーナーの声、資料請求CTA、船カードのレイアウト

### 2. SYH Club（スーパーヨット共同所有）

| 項目 | 内容 |
|------|------|
| URL | https://join.syhclub.com |
| 業態 | スーパーヨットの共同所有クラブ（78名限定） |
| 類似度 | ★★★★★ |

**デザイン分析:**

**取り入れるべき要素:** 限定メンバー数の訴求、イベント/体験のビジュアル化、比較セクション
