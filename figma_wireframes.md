## Figma AI ワイヤーフレーム用プロンプト

### 全体構成

```
Create a 32-page wireframe for a luxury yacht catalog.
Size: A4 portrait (210 x 297mm)
Style: Low-fidelity, grayscale boxes and lines only
No images, no colors - just placeholder boxes marked with X

Grid: 12 columns, 16px gutter, 40px margin
Use gray (#E0E0E0) for image placeholders
Use black lines for text placeholders
Use dashed lines for optional elements
```

### 表紙ワイヤーフレーム

```
Create a wireframe for catalog cover:
- Large rectangle (full bleed) with X mark for hero image
- Centered text placeholder below: 2 lines
- Thin horizontal line at bottom
- Page number indicator: "1"
- A4 size, grayscale only
```

### スプレッドワイヤーフレーム

```
Create a double-page spread wireframe:
- Left page: Large text placeholder (3 lines, centered)
- Right page: Full rectangle with X for image
- Small caption line at bottom right
- Page numbers: "2" and "3" at outer corners
- Grayscale, low-fidelity style
```

### プロダクト特集ワイヤーフレーム

```
Create a 4-page product feature wireframe:
Page 1: Full-bleed image placeholder + title line
Page 2-3: Spread with large image + floating specs card (white rectangle with 4 text lines)
Page 4: 2x3 grid of square image placeholders + section title
All grayscale, boxes marked with X for images
```

### カタロググリッドワイヤーフレーム

```
Create a catalog grid wireframe:
- 3 columns × 5 rows
- Each cell: Square placeholder (X mark) + 2 text lines below
- Section title at top
- Generous spacing between items
- Grayscale, low-fidelity
```
