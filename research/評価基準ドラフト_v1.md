# KOGEI CODE 評価基準ドラフト v1

> 作成日: 2026-04-15
> ステータス: ドラフト（次回MTGで議論）
> 参考: PSA社モデル + 経済産業省伝統工芸品認定5項目

---

## 設計方針

- 経済産業省の5項目をベースに、市場価値に直結する項目を追加
- 最初は**合計100点満点**（10カテゴリ x 各10点）
- 4人（植村・伊石・大城・長宮）でトライアル評価 → AIに学習させる
- 完璧な公平性より、**一貫した自社基準**を優先（パーカーポイント方式）

## 評価10カテゴリ（各10点、合計100点）

### A. 経済産業省5項目ベース（50点）

#### 1. 製造過程の手工業性（10点）
> 経産省: 「製造過程の主要部分が手工業的であること」

| 点数 | 基準 |
|------|------|
| 10 | 全工程100%手作業、機械補助なし |
| 8-9 | 主要工程は手作業、一部軽微な機械補助のみ |
| 6-7 | 手作業が主だが、成形等に機械使用 |
| 4-5 | 手作業と機械の併用 |
| 1-3 | 機械生産が主、仕上げのみ手作業 |

#### 2. 伝統的技術・技法（10点）
> 経産省: 「伝統的な技術又は技法により製造されるものであること」

| 点数 | 基準 |
|------|------|
| 10 | 100年以上の伝統技法を完全に継承、文献・師弟関係で証明可能 |
| 8-9 | 伝統技法をベースに現代的な微調整あり |
| 6-7 | 伝統技法を参考にしつつ独自の解釈を加えている |
| 4-5 | 伝統的な様式だが技法は現代的 |
| 1-3 | 伝統工芸の形式的な模倣 |

#### 3. 伝統的原材料（10点）
> 経産省: 「伝統的に使用されてきた原材料が主たる原材料であること」

| 点数 | 基準 |
|------|------|
| 10 | 伝統的原材料100%、産地・供給元が明確 |
| 8-9 | 主原材料は伝統的、一部代替材（品質維持目的） |
| 6-7 | 伝統的原材料を主に使用、一部現代材 |
| 4-5 | 伝統的原材料と現代材が混在 |
| 1-3 | 主に現代材、伝統的原材料は一部のみ |

#### 4. 地域産業としての実績（10点）
> 経産省: 「一定の地域で産地を形成していること」

| 点数 | 基準 |
|------|------|
| 10 | 経産省指定産地の中核的工房、後継者育成に積極的 |
| 8-9 | 指定産地内の工房、産地組合に所属 |
| 6-7 | 指定産地の近隣、技法的つながりあり |
| 4-5 | 産地外だが、師弟関係で技法を継承 |
| 1-3 | 独学・独自技法、産地との関連薄い |

#### 5. 日常生活での使用性（10点）
> 経産省: 「主として日常生活の用に供されるものであること」

| 点数 | 基準 |
|------|------|
| 10 | 実際に使用可能で、使うほど味が出る（漆器、陶器等） |
| 8-9 | 使用可能だが、主に観賞用途が想定される |
| 6-7 | 飾り・インテリアとして日常に溶け込む |
| 4-5 | 純粋な芸術品・コレクターズアイテム |
| 1-3 | 特殊用途（祭礼・儀式用等）、日常使用不可 |

### B. KOGEI CODE独自項目（50点）

#### 6. 希少性（10点）

| 点数 | 基準 |
|------|------|
| 10 | 作者が唯一の継承者、年間生産数1桁 |
| 8-9 | 継承者5人以下、年間生産数10点以下 |
| 6-7 | 継承者10人以下、限定生産 |
| 4-5 | 一定数の職人が生産、流通量中程度 |
| 1-3 | 大量生産品、容易に入手可能 |

#### 7. 品質・仕上がり（10点）

| 点数 | 基準 |
|------|------|
| 10 | 欠点なし。均一性・精密さ・仕上げすべて完璧 |
| 8-9 | ごく軽微な個体差のみ。プロの目で判別するレベル |
| 6-7 | わずかな不均一・小さな傷。一般には気づかない |
| 4-5 | 目視で確認できる軽微な品質差 |
| 1-3 | 明確な品質問題がある |

#### 8. 作者の実績・受賞歴（10点）

| 点数 | 基準 |
|------|------|
| 10 | 人間国宝（重要無形文化財保持者）、または同等の国際的評価 |
| 8-9 | 伝統工芸士認定 + 全国展受賞歴複数 |
| 6-7 | 伝統工芸士認定、または地域展受賞歴 |
| 4-5 | 修行10年以上、展覧会出品歴あり |
| 1-3 | 新進作家、実績構築中 |

#### 9. 海外市場適合性（10点）

| 点数 | 基準 |
|------|------|
| 10 | 海外需要が確認済み（eBay等で取引実績）、輸送リスク低い |
| 8-9 | 海外コレクター市場あり、輸出制限なし |
| 6-7 | 海外での認知度あり、輸送に注意が必要 |
| 4-5 | 日本国内市場が主、海外未開拓 |
| 1-3 | 輸出困難（法規制・サイズ・壊れやすさ） |

#### 10. ストーリー性・文化的価値（10点）

| 点数 | 基準 |
|------|------|
| 10 | 歴史的エピソード・文化的意義が豊富で、物語としても魅力的 |
| 8-9 | 産地・技法の歴史が深く、解説資料が充実 |
| 6-7 | 一定の歴史的背景あり、説明可能 |
| 4-5 | 歴史的背景は薄いが、現代的なストーリーあり |
| 1-3 | ストーリー性に乏しい |

## 運用フロー（Phase 1）

```
1. 職人/コレクターから鑑定依頼
   ↓
2. 写真提出（5方向: 正面・裏面・左右・底面 + 詳細部分）
   ↓
3. 4人で独立評価（10カテゴリ各10点）
   ↓
4. 平均スコア算出 → KGグレード決定
   ↓
5. 鑑定証明書発行（QRコード付き）
   ↓
6. オンラインDBに登録（証明書番号で検索可能）
   ↓
7. 希望者にはeBay出品代行
```

## 鑑定料金案（Phase 1）

| コース | 料金 | 納期 | 対象 |
|--------|------|------|------|
| スタンダード | 3,000円 | 14営業日 | 一般的な工芸品 |
| プレミアム | 8,000円 | 5営業日 | 急ぎ・高額品 |
| コレクター会員 | 2,000円/件 | 14営業日 | 年会費12,000円で割引 |
