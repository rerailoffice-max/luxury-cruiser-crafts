# マリーナ調査報告書 — 全国版（超富裕層向け大型クルーザー対応施設）

> **調査目的:** 日本全国において、1億円以上の大型クルーザー（40ft以上）が停泊・体験できるマリーナを調査し、カタログ制作およびビジネス展開の基礎資料とする。沖縄はリゾートレジャーの観点から重点調査。
> **初版調査日:** 2026年3月3日（関東版）
> **全国版更新日:** 2026年3月5日

---

### 主要な発見

1. **スーパーヨット（80ft+）に対応できる施設は全国で5施設**
   - 横浜ベイサイドマリーナ（最大144ft / 44m）← 関東
   - 与那原マリーナ（最大164ft / 50m）← 沖縄
   - ラグナマリーナ（最大100ft）← 中部
   - ハウステンボスマリーナ（最大50ft+）← 九州
   - スーパーヨットベース神戸（200ft+ / 2027年開業予定）← 関西

2. **沖縄は「レジャー×ラグジュアリー」の最強エリア**
   - 与那原マリーナは最大50mのスーパーヨット対応 + 74ftチャータークルーザー常駐
   - 慶良間・宮古・石垣への島巡りルートが富裕層の体験価値を最大化
   - 年間を通じたマリンアクティビティが可能（水温20℃以上）

3. **関西はスーパーヨット誘致の国策が追い風**
   - 神戸港に2027年春「SUPERYACHT BASE KOBE」が開業予定（200ft対応）
   - 大阪万博レガシー + IR構想でスーパーヨット受入体制が急速に整備中
   - 瀬戸内海クルーズという独自の体験価値

4. **Azimut Grande 27M（88ft / 26.78m）の停泊可能施設**
   - **横浜ベイサイドマリーナ** — M区画（24〜28m）で収容可能
   - **与那原マリーナ** — 最大50m対応、余裕で収容
   - **ラグナマリーナ** — L桟橋で100ftまで対応
   - **スーパーヨットベース神戸** — 75ft〜対応（2027年〜）

## 4. 沖縄エリア ★重点調査

> **日本屈指のリゾートレジャーエリア。** 年間を通じたマリンアクティビティ、慶良間諸島国立公園の透明度世界トップクラスの海、宮古・石垣への島巡りルートが、富裕層の「体験型ラグジュアリー」に最適。

### 沖縄エリア概要

| 強み | 内容 |
|------|------|
| **気候** | 年間平均気温23℃、水温20℃以上を通年キープ。冬でもマリンアクティビティ可能 |
| **海の透明度** | 慶良間諸島は「ケラマブルー」と呼ばれ、世界有数の透明度（30m+） |

## 7. 九州エリア（福岡・長崎）

> **ハウステンボスマリーナは日本初の本格海上係留マリーナ。** 五島列島・九十九島への雄大なクルーズルートが魅力。

---

### 7-1. ハウステンボスマリーナ ★九州最重要

| 項目 | 内容 |
|------|------|
| **所在地** | 長崎県佐世保市ハウステンボス町 |
| **電話** | 0956-27-0258 |
| **URL** | https://htb-marina.com/ |
| **Google Maps** | [地図を開く](https://www.google.com/maps/search/ハウステンボスマリーナ+長崎県佐世保市) |
| **メール** | marina@htb-marina.com |

**施設規模:**
- **日本初の本格的海上係留方式マリーナ**
- 海上係留: **140隻**（30〜50ftクラス）
- バースサイズ: 30ft / 35ft / 40ft / 50ft
- 35tクレーン（自走式）
- 給油施設（ガソリン・軽油・免税軽油）
- 給水・給電設備
- クラブハウス完備

**施設の様子:** ハウステンボスのヨーロッパ風の街並みを背景に、大村湾の穏やかな入り江にクルーザーが並ぶ。オランダの港町を思わせる風景。干満差が少なく波が立たないため、保管安全性に優れる。

**クルージングルート:**
- **九十九島** — 208の島々が点在する絶景海域（マリーナから約30分）
- **五島列島** — 世界遺産の教会群、美しい海岸線
- **平戸** — 歴史と異国情緒の町

**評価:**
- 50ftまでの制約、Grande 27M（88ft）は超過で停泊不可
- ハウステンボスのリゾート環境との連携は強力
- 九十九島・五島列島クルーズは唯一無二の体験価値
- EXULT 43（43ft）との相性は抜群
- **九州展開の拠点マリーナとして推薦**
