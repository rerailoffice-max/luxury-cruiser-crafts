## カテゴリ別ランキング（参入推奨順）

### 1位: 浮世絵 ★★★★★（最推奨）

| 項目 | 内容 |
|------|------|
| 価格帯 | 普及品$20-500、明治期$500-15,000、江戸期名品$5,000-500,000+ |
| 鑑定効果 | **高い** — 来歴証明・状態レポートで信頼性UP |
| 競合 | 激しいが、新版画（shin-hanga）は20年で400-1,000%値上がり |
| 参入理由 | **軽量・薄型で送料安い、破損リスク低い、鑑定の付加価値高い** |

### 2位: 陶磁器（九谷焼・有田焼・薩摩焼） ★★★★

| 項目 | 内容 |
|------|------|
| 価格帯 | 小品$50-300、良品$300-5,000、名品$5,000-50,000+ |
| 鑑定効果 | **高い** — 窯元・作家の特定で大幅な価格差 |
| 競合 | 多いが、**2023年以降価格が約40%上昇中**で追い風 |
| 注意 | 破損リスク → ダブルボックス梱包必須（+$15-30/個） |

### 3位: 刀剣 ★★★

| 項目 | 内容 |
|------|------|
| 価格帯 | 手打ち$1,500-10,000、本物$10,000-500,000+ |
| 鑑定効果 | **極めて高い** — NBTHK鑑定書で**200-600%プレミアム**。偽物70-80%のため必須 |
| 競合 | 激しいが、信頼できる鑑定者の価値が極めて高い |
| 注意 | 輸出許可証必須（40,000円/本）、文化財は輸出禁止、米国向けEMS停止中 |

### 4位: 漆器・蒔絵 ★★★

| 項目 | 内容 |
|------|------|
| 価格帯 | 一般品$40-450、アンティーク蒔絵$500-5,000+ |
| 鑑定効果 | **中〜高** — 時代・技法の専門鑑定が差別化要因 |
| 競合 | やや低い（参入障壁: 専門知識が必要） |

### 5位: 茶道具 ★★★

| 項目 | 内容 |
|------|------|
| 価格帯 | セット$80-150、アンティーク茶碗$200-10,000+ |
| 鑑定効果 | **中〜高** — 名窯の茶碗は鑑定必須 |

### 6位: 甲冑 ★★

| 項目 | 内容 |
|------|------|
| 価格帯 | レプリカ$320-3,298、本物$3,000-15,000+ |
| 鑑定効果 | 中 |
| 注意 | 大型重量物。送料$100-300 |

---

## 避けるべきカテゴリ

| カテゴリ | 理由 |
|---------|------|
| こけし | 単価低い（$28-190）、19,000件超で飽和、鑑定コストに見合わない |
| 招き猫 | 中国製廉価品が大量、差別化困難 |
| 盆栽 | 植物検疫法により日本→海外発送がほぼ不可能 |
| 切子 | ニッチすぎて回転率低い |

---

## 推奨参入戦略（3人チーム）

### Phase 1（0-6ヶ月）: 浮世絵 + 陶磁器

- 浮世絵: 低リスク・低送料で回転率を上げる
- 陶磁器: 骨董市で仕入れやすい。九谷焼・薩摩焼の知識蓄積
- **鑑定書フォーマット確立 + ブランド認知構築**

### Phase 2（6-12ヶ月）: 刀剣・茶道具に拡大

- Phase 1の信頼・レビューを活かし高単価カテゴリへ
- NBTHK等の既存鑑定機関との連携
- 刀剣の輸出許可プロセス確立

### Phase 3（12ヶ月以降）: 漆器・蒔絵、甲冑

- 専門知識が必要なニッチで差別化

## 鑑定証明書の価格インパクト順

1. **刀剣**: 200-600%プレミアム（最大効果）
2. **陶磁器**: 窯元・作家特定で2-5倍の差
3. **浮世絵**: 来歴・状態証明で大幅UP
4. **漆器・蒔絵**: 時代・技法鑑定が差別化要因
5. **茶道具**: 名窯の茶碗は鑑定必須
