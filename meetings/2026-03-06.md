## 1. 会議で確定した意思決定事項

会議中に以下の重要な方針が3名の合意で決定された。

| # | 決定事項 | 詳細 |
|---|---------|------|
| **1** | **Phase 1 は共同所有型（A案）で即スタート** | ファンド型（B案）の第二種金融商品取引業は時間・コストがかかりすぎるため、まずはA案で認知度を上げる |
| **2** | **船舶はオールジャパン製にこだわる** | Azimutなどヨーロッパ船からLexus LY680 / EXULT 43クラスの日本製に方針転換。ドバイのホテルがヨーロッパ製品で埋め尽くすように、「すべて日本製」を徹底 |
| **3** | **サイズはLexusクラス（20m前後）が最適** | 5〜10人が乗れるサイズ感。YOSHIKIが一級小型免許で操船可能な24m未満。頻繁にパーティーはしないので、使いやすいサイズ重視 |
| **4** | **投資リターンより体験価値を優先** | 年率10%は現実的に難しい。NOT A HOTELも年率5%程度。体験価値をメインに、プラスアルファで投資リターンがあれば嬉しい、というポジショニング |
| **5** | **メインターゲットは医師（特に美容外科・歯科）** | リサーチでは中小企業経営者が最大母数だが、会議では伊石氏のジェノバ人脈を活用し、医師向けに先行営業する方針 |
| **6** | **クローズドな限定募集からスタート** | 広告を打たず、人脈ベースで共同オーナーを集める。「ここに入ること自体が価値」というクラブ型 |
| **7** | **体験コンテンツを充実させる** | 寿司職人の船上握り、和牛ショー（和牛マフィア的な演出）、各種オプションプランで付加価値を上げる |
| **8** | **工芸コードブランドとしての物販も検討** | クルーザー内に配置した工芸品を「KOGEI CODEセレクション」として販売する。クラファン物販とも相性良い |

## 3. 会議で新たに浮上した論点

### 体験価値 vs 投資リターン

会議の最も重要な議論。3名の合意は以下の通り:

```
体験価値をメイン ──── 投資リターンはプラスアルファ

・NOT A HOTELでも年率5%程度
・10%は非現実的
・承認欲求を満たすサービスとして設計
・「社長としてクルーザーオーナー」というステータス
・オプションサービス（寿司職人、和牛等）で追加収益
```

### オールジャパン戦略

伊石氏の提案で全員合意。ドバイの高級ホテルがヨーロッパ製品で統一するように、**船体から包丁・ボールペン1本に至るまで全て日本製**にする方針。

これにより:
- 船舶: Lexus LY680 / Yamaha EXULT 43 / Toyota Ponam（日本メーカー限定）
- 内装: 日本の伝統工芸で統一
- 食器: 有田焼、江戸切子等
- アメニティ: 日本製のみ
- ストーリー: 「オールジャパンメイド」のブランド価値

### 医師向け営業チャネル（新規）

伊石氏が具体的な営業ルートを提案:
- **ジェノバ（プライム上場）** の元社員人脈を活用
- ジェノバはメディカルドック（医師向けプラットフォーム）を運営
- 医師向け営業に特化したリストを保有
- 「医師は褒められると金を出す」（伊石氏の実体験ベース）
- 美容外科・歯科医がメインターゲット

### 工芸コードブランドの物販構想

植村氏の提案:
- クルーザー内の工芸品を「KOGEI CODEセレクション」として一般販売
- クラウドファンディング物販と相性が良い
- 年間維持費の一部をこの物販収益で賄う構想
