# クルーザー投資ファンド ターゲット市場調査レポート

> **調査目的:** 一級小型船舶操縦士免許で操船可能なクルーザーの最大ユーザー層を特定し、投資家ファンド募集事業のターゲット戦略を策定する。
> **調査日:** 2026年3月6日
> **関連資料:** [クルーザー投資事業_スキーム調査報告書.md](クルーザー投資事業_スキーム調査報告書.md) / [クルーザー購入_リフィット調査報告書.md](クルーザー購入_リフィット調査報告書.md)

---

## 1. エグゼクティブサマリー

### 結論

一級小型免許で操船可能なクルーザー（24m未満）において、**最もユーザー数が多いターゲットは「年商1億〜50億円の中小企業オーナー経営者」**である。

| 項目 | 内容 |
|------|------|
| **メインターゲット** | 中小企業オーナー経営者（年商1億〜50億） |
| **年齢** | 45〜65歳 |
| **年収** | 2,000万〜1億円 |
| **純金融資産** | 1億〜5億円 |
| **購入動機1位** | 節税（法人名義で4年償却） |
| **購入動機2位** | 接待（東京湾・大阪湾でのビジネスクルージング） |
| **購入動機3位** | ステータス・福利厚生 |
| **ボリュームゾーン** | 30〜45ft（9〜14m）/ 購入価格 2,000万〜1.5億円 |
| **年間維持費** | 100万〜500万円 |
| **利用ハイシーズン** | 5月〜9月（稼働率60〜80%） |

### ファンド事業への示唆

共同所有型（スキーム調査報告書 A案）の投資家募集において、**税理士・会計士チャネル経由での中小企業経営者へのアプローチ**が最も効率的。初期4〜6名の投資家は経営者人脈からの直接募集が現実的。

## 2. 一級小型船舶操縦士免許の操船範囲

### 免許で操船可能な船舶

| 条件 | 範囲 |
|------|------|
| **基本** | 総トン数20トン未満の船舶 |
| **特例** | 長さ24m未満のプレジャーボート（20トン以上でも可） |
| **航行区域** | 制限なし（外洋含む） |
| **年齢要件** | 18歳以上 |

**重要:** スキーム調査報告書で検討中の Azimut 80（24.38m）および Grande 27M（26.78m）は24mを超過するため、一級小型免許の範囲外となる可能性がある。ただし、総トン数が20トン未満であれば操船可能。実際の総トン数は個体ごとに異なるため、購入前にJCI登録情報の確認が必須。

### 法人名義購入の節税メカニズム

クルーザー所有者の大多数が法人名義で購入している現実がある。マリーナに係留されているクルーザーの所有者は「ほとんどが法人」と言われている。

| 項目 | 内容 |
|------|------|
| **法定耐用年数** | モーターボート: 4年 / ヨット（帆船）: 5年 |
| **中古の簡便法** | 耐用年数の全部を経過 → 法定の20% = 最短2年 |
| **償却方法** | 定率法（初年度に大きく償却）or 定額法 |

**具体例: 1.5億円のクルーザーを法人名義で購入（定率法・4年）**

| 年度 | 償却率 | 年間償却額 | 累計償却額 | 帳簿価額 |
|------|--------|----------|----------|---------|
| 1年目 | 50.0% | 7,500万円 | 7,500万円 | 7,500万円 |
| 2年目 | 50.0% | 3,750万円 | 1.125億円 | 3,750万円 |
| 3年目 | 50.0% | 1,875万円 | 1.3125億円 | 1,875万円 |
| 4年目 | — | 1,874万円 | 1.5億円 | 1円 |

**初年度だけで7,500万円の経費計上が可能。** 法人税率約30%とすると、初年度の節税効果は約2,250万円。

### 維持費の経費処理

| 費目 | 処理科目 | 損金算入 | 条件 |
|------|---------|---------|------|
| マリーナ保管料 | 福利厚生費 | 全額可 | 福利厚生規程の整備、従業員への公平な利用機会 |
| メンテナンス費 | 福利厚生費 | 全額可 | 同上 |
| 燃料費 | 福利厚生費 | 全額可 | 乗船名簿の作成・保管 |
| 保険料 | 福利厚生費 | 全額可 | — |
| 接待利用 | 交際費 | 年800万円まで（中小企業） | 接待記録の保管 |

**注意:** 税務調査での否認リスクは高い。社長・役員の個人利用が大部分の場合、「役員賞与」として認定される可能性がある。福利厚生として認められるには、従業員の実際の利用実績が必須。

## 5. サブターゲット層

### A. 開業医・医療法人理事長

| 項目 | 内容 |
|------|------|
| **推定対象世帯数** | 約26万世帯（富裕層の16%） |
| **平均年収** | 勤務医: 約1,467万円 / 開業医: 約2,690万円 |
| **純金融資産** | 1億〜3億円 |
| **購入動機** | ストレス発散、ステータス、医師仲間との交流、節税 |
| **特徴** | 可処分時間が限られるため「手間ゼロ」の運営代行が刺さる |
| **アプローチ** | 医師向け資産運用メディア、医師専門FP、学会スポンサー |

### B. 不動産オーナー・地主

| 項目 | 内容 |
|------|------|
| **推定対象世帯数** | 約20万世帯（富裕層の12%） |
| **年間キャッシュフロー** | 2,000万〜1億円（家賃収入） |
| **純金融資産** | 1億〜10億円（不動産含む総資産はさらに大） |
| **購入動機** | 資産の分散、新しい体験、不動産とは異なる減価償却資産 |

### 企業接待クルージングの詳細

法人利用が最大の需要ドライバーである。東京湾クルーザーチャーター市場では、累計10,000件以上の法人クルーズ実績（ビズクル社の例）があり、NTT DATA、電通、Cisco、Adobeなどの大手企業が利用。

| 項目 | 内容 |
|------|------|
| **利用目的** | 重要顧客接待、海外投資家との商談、幹部懇親会 |
| **参加人数** | 6〜15名が中心 |
| **時間** | 3〜6時間（ディナークルーズが主流） |
| **エリア** | 東京湾（お台場〜レインボーブリッジ〜横浜）が最大市場 |
| **料理** | ケータリングまたは船上シェフ（1名あたり8,000〜15,000円） |
| **チャーター相場** | 半日20万〜50万円 / 終日40万〜80万円 / プレミアム100万円以上 |

**接待クルージングの優位性:**
- プライベート空間で機密性の高い商談が可能
- 東京湾の夜景が強い印象を与える
- 料理・ドリンクのカスタマイズで相手の嗜好に対応
- 「他にない特別な体験」が差別化になる

### 自社クルーザーと共同所有の価値

チャーター（都度レンタル）と比較した場合の自社/共同所有の価値:

| 比較項目 | チャーター | 共同所有（ファンド） |
|---------|----------|------------------|
| 1回あたりコスト | 20万〜100万円 | 実質5万〜15万円（年会費按分） |
| 予約の自由度 | 空き状況に依存 | 優先予約権あり |
| カスタマイズ | 限定的 | 伝統工芸内装で差別化 |
| ステータス | 借りている | 「オーナー」の肩書き |
| 節税効果 | なし | 減価償却による法人節税 |
| 資産性 | なし | 持分売却が可能 |

### 年間稼働日数の推定

| 所有形態 | 年間稼働日数 | 稼働率 |
|---------|------------|--------|
| 個人単独所有 | 15〜30日 | 4〜8% |
| **共同所有（4〜6口）** | **40〜80日** | **11〜22%** |
| チャーター運用併用 | 80〜120日 | 22〜33% |

**稼働率向上のポイント:**
- 共同所有によりオーナー利用日を分散 → 単独所有の2〜3倍の稼働
- オーナー未使用日をチャーター運用 → 収益化とさらなる稼働率向上
- オフシーズン対策: 冬季ディナークルーズ、忘年会パッケージの企画

### 御社の差別化ポイント

```
                 ステータス・体験価値
                      高い
                       ↑
                       │
          Yacht Life   │  ★ 御社（計画）
          Design ●     │  ● ← 「伝統工芸 × 節税 × 共同所有」
                       │     の三位一体で唯一のポジション
  小規模 ←─────────────┼─────────────→ 大規模
                       │

                       │

                       ↓
                      低い
```

**御社の独自ポジション:**
1. **伝統工芸内装** — 日本の匠の技で内装リフィットした唯一無二のクルーザー
2. **節税スキーム** — 法人名義の共同持分で減価償却のメリットを投資家に提供
3. **資産価値** — 伝統工芸品としての付加価値で、通常のクルーザーより減価が緩やか
4. **文化的ストーリー** — 海外富裕層への接待で「日本文化」を体感させる強力なツール

## 10. ファンド投資家募集戦略

### Phase 1: 共同所有型（最初の4〜6名）

スキーム調査報告書のA案に基づく。第二種金融商品取引業の登録不要で即座に開始可能。

#### 募集チャネル優先順位

| 優先度 | チャネル | 具体的アクション | 期待効果 |
|--------|---------|---------------|---------|
| **1位** | **税理士・会計士紹介** | 節税に積極的な顧問税理士を持つ経営者を紹介してもらう | 節税ニーズが確実、成約率が最も高い |
| **2位** | **経営者コミュニティ** | EO、YPO、ロータリークラブ、経営者合宿等での紹介 | 口コミによる信頼性、同質性の高いグループ |
| **3位** | **マリーナ/ボートショー** | ジャパンインターナショナルボートショー出展、マリーナでの試乗会 | 既に船への関心があるターゲット |
| **4位** | **富裕層向けメディア** | PRESIDENT、Forbes Japan、Nikkei Luxeへの記事掲載/広告 | 認知拡大、ブランディング |
| **5位** | **医師向けチャネル** | 医師専門資産運用セミナー、ケアネット等の医師メディア | サブターゲットへのアプローチ |

#### 投資家に訴求するキーメッセージ

| ターゲット | 刺さるメッセージ | 具体的な数字 |
|-----------|----------------|------------|
| 経営者（節税重視） | 「1.5億円のクルーザーで初年度7,500万円の経費計上」 | 節税効果: 初年度約2,250万円 |
| 経営者（接待重視） | 「東京湾の夜景をバックに、他にない接待空間」 | チャーター相場の1/3〜1/5のコスト |
| 医師 | 「手間ゼロ、予約だけ。オーナーの肩書きと非日常体験」 | 年12回利用で1回あたり実質25万円 |
| 不動産オーナー | 「不動産とは異なる4年償却の減価資産」 | 不動産（47年）vs クルーザー（4年） |

### Phase 2: ファンド型への拡大

スキーム調査報告書のB案。第二種金融商品取引業を取得後、小口化して広く投資家を募集。

| 項目 | 内容 |
|------|------|
| **最小投資単位** | 500万〜1,000万円 |
| **想定投資家数** | 10〜30名/隻 |
| **チャーター運用** | オーナー未使用日を外部チャーターに開放 |
| **想定利回り** | 年3〜8%（チャーター収入+船体売却益） |
| **セカンダリー** | 持分の二次売買プラットフォームで流動性確保 |

### 口コミ循環モデル

最も効果的な募集は既存投資家からの紹介。以下の循環を設計する。

```
  投資家が乗船体験
        │
        ↓
  「これはすごい」と感動
  （伝統工芸内装 × 東京湾夜景）
        │
        ↓
  経営者仲間を次回クルーズに招待
  （試乗体験会としてフリー招待）
        │
        ↓
  招待された経営者が
  節税メリット + 体験価値に納得
        │
        ↓
  新規投資家として参画
        │
        ↓
  さらに次の経営者仲間を招待...
```

**KPI目標:**
- 既存投資家1名あたり年間2名の紹介を目標
- 試乗体験会からの成約率: 20〜30%

## 13. まとめ: ファンド事業の勝ち筋

### 最優先ターゲット

**年商3億〜30億円の中小企業オーナー経営者（45〜60歳）** が最大のターゲット層。

理由:
1. 富裕層の57%を占める最大勢力
2. 法人名義購入の節税メリットを最大限に享受できる
3. 接待需要があり、事業上の実利がある
4. 経営者同士のネットワークで口コミが広がる
5. 意思決定が速い（個人オーナーのため稟議不要）

### 最適な商品設計

| 項目 | 推薦 |
|------|------|
| **対象クルーザー** | 40〜50ft（12〜15m）の日本メーカー艇（EXULT 43クラス） |
| **理由** | ①一級小型免許で確実に操船可能 ②輸入不要でコスト削減 ③国内メンテナンス充実 ④伝統工芸との相性◎ |
| **共同口数** | 4口（1口あたり月3〜4日利用） |
| **1口あたり初期投資** | 3,000万〜5,000万円 |
| **1口あたり年会費** | 200万〜300万円 |
| **スキーム** | A案（共同所有型） → Phase 2でB案（ファンド型）へ移行 |

### 次のアクション

1. **税理士ネットワークの構築** — 節税に積極的な税理士3〜5名とのアライアンス
2. **試乗体験会の企画** — チャータークルーザーで仮の体験会を実施、反応を検証
3. **投資家向け資料の作成** — 本レポートをベースに、節税シミュレーション付きの提案書を作成
4. **法人設立・船舶貸渡業届出** — スキーム調査報告書のPhase 1に準じて実行
